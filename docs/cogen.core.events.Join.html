Class: cogen.core.events.Join

A operator for waiting on a coroutine. Example:

@coroutine
def coro_a():
    return_value = yield events.Join(ref)


@coroutine
def coro_b():
    yield "bla"
    raise StopIteration("some return value")

ref = scheduler.add(coro_b)
scheduler.add(coro_a)
